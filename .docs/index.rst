dynml
=====

This repository contains all code related to the ``dynml`` package. In this
python package, we implement various concepts from dynamical systems and
machine learning. Please see
:doc:`/test_dynml/examples </rsts/test_dynml/examples/__init__>` and 
:doc:`/test_dynml/examples </rsts/dynml/examples/__init__>` for example code
and documentation, respectively.

Modules found within the ``dynml`` package were tested using ``pytest``, type
checked using ``mypy``, style checked using ``flake8``, documented, and
commented.
